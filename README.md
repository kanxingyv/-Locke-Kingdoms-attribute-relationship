# 洛克王国属性克制关系图

一个展示洛克王国属性克制关系的交互式网页应用。

## 功能特点

- 🎯 18 种属性圆形布局展示
- 🔄 点击属性图标查看克制关系
- 📊 四大克制关系分类展示:
  - 我克制 (红色)
  - 我抵抗 (青色)
  - 克制我 (绿色)
  - 抵抗我 (粉色)
- ⚔️ 天敌关系特别标记
- 📱 响应式设计，完美适配移动端

## 使用方法

1. 直接双击 `index.html` 文件在浏览器中打开
2. 或者使用本地服务器:
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx http-server -p 8000
   ```
3. 访问 `http://localhost:8000`

## 操作说明

- 点击圆环上的任意属性图标
- 中间圆形会显示选中的属性
- 下方展示该属性的详细克制关系
- 红色标记表示天敌关系
